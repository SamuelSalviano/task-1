# task-1

há dois comandos que são utilizados,o "npm install" e o "npm run dev". O "npm install" serve para nos instalarmos pelo terminal as dependencias e as devDependencias (Como por exemplo: o express, typescript e o ts-node)
Já o "npm run dev" é um script que é rodado pelo tsx (dependencia instalado pelo comando "npm install" no terminal) com o comando "watch" fazendo assim o "server.ts" rodar e aparecer na tela.
